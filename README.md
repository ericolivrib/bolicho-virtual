# Bolicho Virtual

Projeto semestral de **Programação Orientada a Objetos para Web I** focado em um sistema de venda de produtos coloniais e artesanais.

### Tecnologias que serão utilizadas

- Java
- Maven
- Servlet
- JSP  
- Bootstrap
- PostgreSQL
